# collection

先設定一個陣列名為`list`

```javascript
>> var list = [1,2,3]
< undefined
/* 函數名稱 */
>> list
< Array(3) [1,2,3]

(3) […]

/*1. key: value*/
0: 1
1: 2
2: 3


length: 3

<prototype>: Array [] /* 原型是prototype */
```

1.`key[value]`

```javascript
list[0]
1
```

2.`.length`

```javascript
list.length
```

3.`.indexOf`函數

可以使用`list.indexOf`這個函數，  
是因為在`prototype`裡面有定義此函式。

```javascript
list.indexOf(1)
0
list.indexOf(2)
1
list.indexOf(3)
2
```

# 自訂建構函式

我們還記得函數是可以重複利用的SOP，相同的功能寫成函數後就可以重複利用。接下來在Google Console裡寫一個函數試試：

```javascript
>> function Pokemon(name, skill) {
  this.name = name;
  this.skill = skill;
}
< undefined

>> Pokemon 
< function Pokemon()
V
< Pokemon()
​
arguments: null
caller: null
length: 2
name: "Pokemon"
​
prototype: Object { … }
​
<prototype>: function ()
```
